# geometrymath
Сайт не требует установки, работает в браузере. Учащийся может выполнить все задания, затем скачать PDF-отчёт и отправить его учителю для проверки и анализа. В отчёте указаны: имя ученика, дата, общий счёт, прогресс по каждой фигуре (тест и задачи в абсолютных и процентных значениях).
